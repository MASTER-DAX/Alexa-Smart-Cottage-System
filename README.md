This repository is a compilation of the features and pages of both the web application and mobile application for the Smart Cottage System. It includes the user interfaces, functionality, and layouts used across both platforms.

It also includes the backend system, frontend development tools, software utilities, and design tools used in building and completing the entire system, from development to deployment.
